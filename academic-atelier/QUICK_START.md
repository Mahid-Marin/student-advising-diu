## 🔧 Troubleshooting

### Can't access frontend?
```
Check: http://localhost:3000/ is accessible
Kill port: netstat -ano | findstr :3000
Restart: npm run dev
```

### Can't login?
```
Use credentials:
Email: student@example.com
Password: password
```

### Can't see Knowledge Base?
```
1. Login successfully
2. Click "Knowledge Base" in navigation menu
3. Should see 8 items
4. Check browser console (F12) for errors
```

### Services not running?
```
Check terminals for any error messages
Verify ports 3000 and 8080 are not in use
Restart services as needed
```
